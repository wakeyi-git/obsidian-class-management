# 변경 기록

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다.

## 1.17.0 - 2026-07-22

### 추가 — 하이브리드 심화: 실시 차시 확정·행사 일괄·Bases 확장

- **실시 차시 확정** — `실시 차시 확정 (어제까지)` 명령이 어제까지 실시된 차시를 `교육과정/실시 차시/{월}/` 아래 불변 노트로 생성(날짜·교시·과목·단원·학습 내용·시수·성취기준·수업일지 링크). 과거는 재배정되지 않으므로 한 번 만든 노트는 다시 쓰지 않음 — Bases 전면 전환의 걸림돌(쓰기 증폭) 없이 원장을 노트로 축적
  - 플러그인 로드 시 밀린 날이 14일 이하면 자동 확정, 그보다 많으면 안내만 표시
  - 과목 없는 행사 교시·삭제 교시는 수업이 아니므로 제외
- **행사 노트 일괄 만들기** — 학사일정의 모든 행사 노트를 한 번에 생성하는 명령
- **Bases 보기 3종 추가** — 주간학습안내(주차별), 실시 차시(날짜별·과목별·성취기준별), 연계 기록(단원 노트 임베드용 — 이 단원을 링크한 수업일지·과제·근거 자동 수집)
- **단원 진행률** — 수업일지 저장 시 단원 노트의 `taughtHours` 자동 갱신, 통합 단원 Bases 보기에 실시 시수·진행률(%) 열 추가
- 통합 단원 본문에 `6. 연계 기록` 섹션(연계 기록.base 임베드) 추가
- Bases 보기·행사 노트 템플릿을 순수 모듈(entity-notes.ts)로 이동 — 플러그인과 스크립트가 같은 단일 진실 사용

## 1.16.0 - 2026-07-22

### 변경 — 시간표 칸 조작 방식 분리 (좌클릭 보기·우클릭 편집)

- **좌클릭**: 주간 시간표·오늘 패널의 교시를 클릭하면 차시 인스펙터만 열림 (과목 변경 모달이 함께 뜨지 않음)
- **우클릭**: 컨텍스트 메뉴로 바로 수정 — 과목 목록(현재 과목 ✓, 선택 즉시 변경), 수업 삭제(이날만), 변경 제거(원래대로), 진도 고정(고정된 차시는 📌로 표시), 편집…(기존 과목 변경 모달: 직접 입력·사유 포함)
- 빈 칸·삭제된 교시도 우클릭 메뉴에서 과목 선택으로 바로 추가·복원
- 키보드(Enter·Space)는 좌클릭과 동일하게 차시 인스펙터를 열고, 상세 편집은 인스펙터의 `이 교시 편집` 버튼 이용

## 1.15.0 - 2026-07-22

### 추가 — 교육과정-수업-평가-기록 일체화 연계 층

- **차시 승격(수업일지 만들기)** — 차시 인스펙터의 `수업일지 만들기` 버튼으로 진도표 차시를 수업일지 노트로 승격: 통합 단원 검색 → 날짜·교시·시수·학습 내용이 미리 채워진 수업일지 작성 → 저장 시 진도표 비고에 `[[수업일지]]` 위키링크 자동 기록
- **행사 노트** — 차시 인스펙터의 '이날 행사'와 오늘 패널의 '다가오는 일정'에서 행사를 클릭하면 `교육과정/행사/`에 노트 자동 생성·열기(계획·준비물·연계 단원·결과 성찰 틀 포함)
- **과제-단원 연계** — 과제(평가) 모달에 `연계 단원` 선택 추가: 저장 시 frontmatter(curriculumUnitId·Title·Path)와 본문에 단원 위키링크 기록
- **일체화 Bases 보기** — `일체화 Bases 보기 만들기` 명령으로 `교육과정/뷰/`에 5개 .base 생성: 통합 단원(과목별), 수업일지(단원별 차시), 과제(평가 과제·단원 연계만), 행사 일지, 학생부 근거(영역별·검토 대기)
- 통합 단원 노트에 `class-management/curriculum-unit` 태그 추가(Bases 필터용)

## 1.9.0 - 2026-07-21

### 변경

- 이전 표기(`2026-10-15(3)` 단독, `고정 날짜` 헤더)는 그대로 읽히며 다음 재배정 때 새 형식으로 정리됨

## 1.7.0 - 2026-07-21

### 추가

- 진도표 `고정 날짜` 열이 교시까지 지원 — `2026-10-15(3)` 또는 `2026-10-15 3교시`로 적으면 그 차시가 해당 날짜·교시에 정확히 배정 (행사 연계 차시, 외부 강사 수업 대응)

- 시수 2 이상인 교시 고정 차시는 그 교시부터 연속 배정
