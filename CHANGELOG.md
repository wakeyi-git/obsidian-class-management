# 변경 기록

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다.

## 1.0.0 - 2026-07-21

### 주요 기능

- 학생 명단 단일·CSV·TSV 입력, 재적·전출·졸업 상태와 번호 변경
- 출결 상태·사유, 과제 제출 상태·메모, 생활 기록과 학생 위키링크
- 통합 목록·검색·복합 필터·저장된 보기·학생별 타임라인
- 월간·주간 캘린더와 날짜별 빠른 생성
- GTD 할 일, 가정통신문 회신, 일·주·월 루틴 체크리스트
- 분석 보고서, CSV 내보내기, 근거 링크가 있는 학생 피드백과 2026 기재요령 기반 학교생활기록부 영역별 검토 초안
- 창체·교과·행동 전용 구조화 근거, 학급 일괄 입력, 학생·교과별 누락 점검, RAW 검토 상태와 기존 기록 분류 추천
- `AGENTS.md`, `CLAUDE.md`, `AI_WORKFLOW.md` 기반의 선택적 파일 협업
- 다중 학급·학년도·학기, 다음 학기 명단 복사, 읽기 전용 보관
- 데이터 진단, 보관 기간 검토, Vault 내부 백업·비덮어쓰기 복구·레거시 마이그레이션
- 모바일 레이아웃, 키보드 탐색, 스크린리더 레이블, 제한 동시성 색인
- 첨부 교육과정 운영 자료를 반영한 성취기준 기반 통합 단원 설계, 백워드 설계, 일체화 자동 점검
- 차시별 수업 실행·과정중심 평가 증거·피드백·교사 성찰, 운영 시수 및 캘린더 연결
- 학생부 RAW 근거와 통합 단원·수업일지의 ID 및 위키링크 연결, CSV·데이터 진단 확장
- 2022 개정 교육과정 대비 개념기반 탐구학습 연구를 반영한 핵심 아이디어·개념적 렌즈·개념망·스트랜드 설계
- 스트랜드별 일반화와 사실적·개념적·논쟁적 안내 질문, 내용 지식·핵심 기능·평가방법, 분석적 루브릭
- 참여·집중·조사·조직·일반화·전이·성찰 차시 연결과 학생별 개념적 이해·탐구 과정·전이 증거 기록
