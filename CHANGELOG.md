# 변경 기록

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다.

## 1.11.0 - 2026-07-21

### 추가

- 주간 시간표에서 칸을 수정(과목 변경·교시 추가·삭제·복원)하면 **영향받은 과목의 진도 배정이 자동 갱신** — 변경 전·후 과목의 진도표만 다시 계산해 배정 열에 기록하고, 알림에 `진도 재배정`(확인 필요 건수 포함) 표시
- 진도표 노트를 직접 편집한 경우에는 기존처럼 `진도 자동 배정` 명령으로 갱신

## 1.10.1 - 2026-07-21

### 변경

- 진도표 열 순서를 `고정 | 순 | 배정 | …`으로 조정 (📌가 맨 앞). 1.10.0 순서와 그 이전 순서 노트도 계속 자동 인식
- 차시 고정 검색 목록도 위치(📌/배정) — 순·단원·차시 순으로 표기

## 1.10.0 - 2026-07-21

### 변경

- 진도표 열 순서를 `순 | 고정 | 배정 | 단원·영역 | 학습 내용 | 시수 | 성취기준 | 준비물 | 비고`로 변경 — 고정·배정 상태를 표 앞에서 바로 확인
- 기존 열 순서 노트도 헤더를 보고 자동 인식 (다음 재배정 때 새 순서로 정리)
- 차시 고정 검색 목록도 같은 순서(순 — 📌 위치/배정 — 단원·차시)로 표기하고 '고정'·'배정' 접두어 제거

## 1.9.0 - 2026-07-21

### 변경

- 진도표 열 정리: `고정 날짜` → `고정`. 고정된 차시는 `📌` 표식만 남기고 위치는 `배정` 열이 담당 (배정과 다를 때만 `📌 2026-10-15(3)`로 명시해 위치를 잃지 않음)
- `배정` 열에 연도 포함 (`10-15(3)` → `2026-10-15(3)`) — 12월~1월에 걸친 학기에서도 모호하지 않음
- 이전 표기(`2026-10-15(3)` 단독, `고정 날짜` 헤더)는 그대로 읽히며 다음 재배정 때 새 형식으로 정리됨

## 1.8.1 - 2026-07-21

### 개선

- 고정된 차시 강조 표시: 주간 시간표 칸(📌·강조색·툴팁), 칸 편집 모달(고정 차시 안내 배너), 차시 고정 검색 목록(이 교시 고정 행 배경 강조, 다른 곳 고정 행 📌·강조색)

## 1.8.0 - 2026-07-21

### 추가

- 주간 시간표 칸에서 **날짜-진도 고정** — 칸 클릭 → '이 교시에 차시 고정' → 그 과목 진도표의 차시를 검색해 선택하면 해당 날짜·교시에 고정되고 그 과목이 즉시 재배정됨
- 이미 이 교시에 고정된 차시를 다시 선택하면 고정 해제, 다른 곳에 고정된 차시·현재 배정 위치가 목록에 표시됨

## 1.7.0 - 2026-07-21

### 추가

- 진도표 `고정 날짜` 열이 교시까지 지원 — `2026-10-15(3)` 또는 `2026-10-15 3교시`로 적으면 그 차시가 해당 날짜·교시에 정확히 배정 (행사 연계 차시, 외부 강사 수업 대응)
- 배정 우선순위: 날짜+교시 고정 → 날짜 고정 → 순차. 고정 교시에 해당 과목 수업이 없으면 배정하지 않고 "주간 시간표에서 과목을 먼저 배치하세요" 경고
- 시수 2 이상인 교시 고정 차시는 그 교시부터 연속 배정

## 1.6.1 - 2026-07-21

### 수정

- 주간 시간표가 모든 주를 "현재 설정된 학기"의 시간표로 해석하던 문제 수정 — 이제 각 날짜가 속한 학기의 기초시간표·변경 기록으로 해석 (학기 설정을 바꿔도 지난 학기 주간 시간표가 달라지지 않음)
- 칸 클릭 변경·삭제가 날짜가 속한 학기의 기초시간표 노트에 기록되도록 수정 (설정 학기와 무관), 알림에 기록된 학기 표시
- 주간학습안내 생성도 날짜별 학기의 시간표·진도표 사용, 학기 경계 주 지원
- 진도 자동 배정 알림에 대상 학기 명시 (배정은 원래도 설정 학기의 진도표에만 적용됨)

## 1.6.0 - 2026-07-21

### 변경

- 시수 점검의 **실행** 열을 "오늘까지 시간표 기준으로 운영된 시수"로 변경 (시간표 변경·행사·교시 삭제 반영). 기존의 수업일지 완료 차시 집계는 모든 수업을 수업일지로 쓰지 않는 실사용에서 항상 0에 가까워 대체
- 과목이 지정되지 않은 행사 교시는 시간표에 행사명으로 표시하되 교과 시수 집계·진도 배정에서 제외 — 시수 점검표에 행사명이 "기준 없음" 행으로 나타나던 문제 해소

## 1.5.1 - 2026-07-21

### 수정

- 빈 칸에 수업을 추가할 때 드롭다운의 첫 과목(국어 등)을 그대로 두고 저장하면 등록되지 않던 문제 수정 (표시 값과 내부 상태 동기화)
- 과목 선택 목록이 기준 시수 노트를 우선하도록 변경 — 설정의 일반 교과 목록과 학교자율시간 실제 과목명(예: 디지털 놀이터)이 중복 표시되던 문제 해소. 기준 시수·시간표·진도표가 없을 때만 설정의 교과 목록을 사용

## 1.5.0 - 2026-07-21

### 추가

- 주간 시간표 칸에서 **이 교시 삭제** — 기초시간표는 5교시지만 그날만 4교시로 운영하는 경우 대응
- 삭제는 `시간표 변경` 표에 `－` 표식으로 기록되며(`-`·`삭제`도 인식), 삭제된 칸은 빗금과 "수업 없음"으로 표시
- 삭제된 칸 클릭 → 변경 제거로 원래 시간표 복원 또는 다른 과목 재배정
- 삭제된 교시는 진도 배정·편성 시수·주간학습안내에서 제외

## 1.4.0 - 2026-07-21

### 변경

- 시간표 우선순위를 `시간표 변경 > 행사 > 기초시간표`로 조정 — 행사(체육대회·전일 체험학습 등)가 배정된 날에도 칸 클릭으로 특정 교시를 교과·창체로 재배정 가능

## 1.3.0 - 2026-07-21

### 추가
