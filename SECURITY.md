# 보안 정책

## 지원 버전

보안 수정은 최신 정식 버전을 우선 지원합니다.

## 취약점 제보

공개 저장소가 준비되면 GitHub의 **Security → Report a vulnerability**를 사용해 비공개로 제보해 주세요. 학생 이름, Vault 내용, 실제 학교 자료는 첨부하지 말고 익명화된 재현 절차와 영향 범위만 제공하세요.

보안 문제가 아닌 일반 버그와 기능 요청은 공개 이슈를 사용할 수 있습니다.

## 보안 범위

이 플러그인은 네트워크 요청과 텔레메트리를 포함하지 않습니다. Vault 파일 생성·수정·휴지통 이동, 로컬 백업·복구, 사용자가 별도로 허용한 외부 AI 도구의 폴더 접근이 주요 보안 검토 대상입니다.
