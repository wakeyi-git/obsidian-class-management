# 문제 해결

## 출결 표가 표로 보이지 않음

출결 노트의 표 머리글이 `번호 | 학생 | 상태 | 사유` 4열인지 확인합니다. 이전 3열 표나 레거시 JSON 주석은 **백업·복구·마이그레이션**에서 백업 후 정규화할 수 있습니다.

## 학생 링크가 깨짐

**학급·데이터 → 데이터 진단 실행**으로 깨진 `studentPath`와 현재 명단에 없는 학생 번호를 확인합니다. `studentPath`의 정상 형식은 `[[학급운영/학생/01 이름]]`입니다. 번호·이름 변경은 학생 재적 관리의 **변경** 버튼을 사용하세요.

## 목록·캘린더가 갱신되지 않음

원본 노트를 저장한 뒤 잠시 기다립니다. 활동 색인은 Vault 변경 이벤트를 120ms 디바운스합니다. 그래도 갱신되지 않으면 플러그인을 껐다 켜거나 Obsidian의 **Reload app without saving**을 실행합니다.

## 보관된 학급을 수정할 수 없음

보관된 프로필은 의도적으로 읽기 전용입니다. **학급·데이터**에서 현재 학급 보관을 해제하거나 활성 학급을 전환합니다.

## 백업 복구가 기존 파일을 바꾸지 않음

정상 동작입니다. 복구는 누락 파일만 복사하며 기존 경로를 덮어쓰지 않습니다. 손상된 기존 파일은 먼저 이름을 바꾸거나 안전한 위치로 이동한 뒤 복구를 다시 실행하고 두 사본을 직접 비교하세요.

## AI 초안 버튼이 비활성화됨

분석 화면에서 학생 한 명을 선택하고 **AI 협업 설정**에서 기능을 명시적으로 활성화해야 합니다. 플러그인은 외부 AI를 직접 호출하지 않으며 근거가 포함된 로컬 검토 초안만 만듭니다.

## 모바일 표가 좁음

출결·과제·회신 표는 가로 스크롤을 사용합니다. 화면을 가로로 돌리거나 Obsidian의 글꼴 크기를 조절해 보세요. 핵심 대시보드와 관리 화면은 700px 이하에서 단일 열로 전환됩니다.
