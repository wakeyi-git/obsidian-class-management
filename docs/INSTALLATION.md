# 설치와 업데이트

## 커뮤니티 플러그인 설치

커뮤니티 디렉터리 등록 후에는 Obsidian 설정 → 커뮤니티 플러그인 → 탐색에서 **Classroom Manager**를 찾아 설치하고 활성화합니다.

## 수동 설치

GitHub Release의 같은 버전에서 다음 세 파일을 내려받습니다.

- `manifest.json`
- `main.js`
- `styles.css`

Vault의 `.obsidian/plugins/class-management/` 폴더에 세 파일을 넣고 Obsidian을 다시 시작하거나 플러그인을 다시 불러온 뒤 활성화합니다. 모바일에서는 파일 관리자 접근 방식이 운영체제에 따라 다를 수 있으므로 커뮤니티 디렉터리 설치를 권장합니다.

## 첫 실행

1. **학급 공간 초기화** 명령을 실행합니다.
2. 학생 추가 모달에서 한 명을 입력하거나 CSV·TSV 명렬표를 가져옵니다.
3. **백업·복구·마이그레이션** 화면에서 첫 백업을 만듭니다.
4. 실제 자료 입력 전 별도 테스트 Vault에서 출결·과제·기록 흐름을 확인합니다.

## 업데이트

업데이트 전에 유지관리 화면에서 백업을 만드세요. 커뮤니티 플러그인 화면에서 업데이트한 뒤 **백업·복구·마이그레이션**을 열어 레거시 형식 개수를 확인합니다. 마이그레이션 버튼은 변경 대상이 있을 때만 활성화되며 실행 전 새 백업을 자동 생성합니다.
