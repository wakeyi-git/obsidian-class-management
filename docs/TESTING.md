# 테스트 체크리스트

## 자동 검증

```bash
npm ci
npm run check
```

`check`는 데이터 파서 회귀 테스트, TypeScript 엄격 검사, 프로덕션 번들, 릴리스 메타데이터 검증을 실행합니다.

## 수동 Obsidian 검증

- 데스크톱과 모바일에서 플러그인 활성화·비활성화
- 학생 단일 추가, CSV·TSV·한글 Windows CSV 입력
- 출결 5개 상태와 사유, 과제 3개 상태와 메모, 회신 3개 상태
- 학생 번호·이름 변경 후 원본 위키링크 확인
- 통합 목록 검색·복합 필터·저장된 보기와 200건 더 보기
- 월간·주간 전환, 학생·유형 필터, 날짜별 빠른 생성
- GTD 상태 전환, 루틴 날짜별 체크, 가정통신문 수정
- 일·주·월 보고서와 CSV, 익명·비익명 AI 초안
- 창체·교과·행동 개별 근거와 학급 일괄 입력, 저장 전 경고, RAW·검토 완료·초안 제외 전환
- 학생별 창체·행동 및 교과별 누락 점검, 기존 자유서술 기록 추천·수동 구조화
- 통합 단원 설계 저장·수정, 설계 중 상태의 누락 허용과 실행 준비 전 필수 연결 검증
- 성취기준-핵심 이해-수업-평가 준거-피드백-기록 초점 입력 시 연결도 100% 확인
- 차시 계획·실행 완료 저장, 계획 시수 대비 실행 시수, 통합 목록·월간·주간 캘린더 표시
- 학생부 근거에서 통합 단원·차시 선택 후 교과·성취기준·평가요소 자동 연결과 위키링크 확인
- 개념기반 탐구학습 활성화·비활성화와 기존 통합 단원 하위 호환
- 핵심 아이디어·개념 렌즈·개념·스트랜드·일반화·안내 질문·전이 맥락·루브릭 입력 시 탐구 설계 100% 확인
- 개념적 질문 3~5개, 논쟁적 질문 1~2개, 일반화 진술 형식과 일반화·전이·성찰 단계 누락 경고
- 일반화·전이 차시 실행 완료 시 학생 일반화·전이 증거 필수 입력
- 학생별·학급 일괄 근거의 개념적 이해·탐구 과정·전이 증거 저장과 CSV 내보내기
- 학급 전환, 명단 복사, 읽기 전용 보관과 해제
- 데이터 진단, 백업, 누락 파일 복구, 레거시 마이그레이션
- 키보드 Tab·Enter·Space 탐색, 포커스 표시, 스크린리더 레이블
- 320px 모바일 너비와 밝은·어두운 테마

실제 학생 자료 대신 익명화된 테스트 자료를 사용합니다.
