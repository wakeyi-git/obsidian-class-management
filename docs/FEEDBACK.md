# 실사용 피드백 트리아지

> 2단계(실사용 검증 루프)의 처리 대장. **수집은 볼트에서** — 플러그인 `피드백 기록` 명령이 날짜·버전·화면을 자동 스탬프해 볼트 `학급운영/피드백.md`에 쌓는다. 주 단위로 이 표에 옮겨 트리아지하고, 반영 릴리스를 기록한다.

## 처리 대장

|---|---|---|---|---|
| | | | | |

## 트리아지 규칙

- **수정**: 마찰·버그 — 주 단위 patch(1.x.y)로 해소하고 CHANGELOG에 남긴다.
- **보류**: 기능 요청 — [PRODUCT.md §10](PRODUCT.md) 로드맵 또는 [UIUX-PRINCIPLES.md §7](UIUX-PRINCIPLES.md) 백로그로 옮긴다.
- **의도**: 설계 원칙에 따른 동작 — 볼트 노트 항목에 한 줄 답을 남기고 체크한다.
- 목표 시간 미달(출결 1분·기록 15초·주간 리뷰 10분)은 항상 "수정"으로 취급한다.
