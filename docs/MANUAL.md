# Classroom Manager 시기별 사용 설명서

> 학년 흐름을 따라 "지금 무엇을 하면 되는지"만 담은 교사용 설명서입니다. 기능의 설계 배경은 [PRODUCT.md](PRODUCT.md), 화면 지도는 [UX-FLOWS.md](UX-FLOWS.md) 참고. (1.31.0 기준)

## 기본기 — 화면과 조작 (처음 5분)

- **화면 3분할**: 왼쪽 **학급 메뉴**(뷰 이동 + 현재 화면 작업) · 가운데 **뷰** · 오른쪽 **오늘 패널**(오늘 시간표·출결·마감)과 **인스펙터**(학생/차시 상세)
- **조작 규칙 하나만 기억**: **좌클릭(탭) = 보기, 우클릭(길게 누르기) = 편집**. 시간표 교시를 좌클릭하면 차시 인스펙터, 우클릭하면 과목 변경·수업 삭제·진도 고정 메뉴
- **명령 팔레트**(Cmd/Ctrl+P)에서 아래의 모든 `명령`을 이름으로 실행할 수 있습니다
- 모든 데이터는 볼트의 Markdown 노트입니다 — 표를 직접 고쳐도 플러그인이 읽습니다

```
학급운영/
├─ 학생 · 학생 기록 · 출결 · 과제        ← 학생 데이터
├─ 할 일 · 가정통신문 · 루틴             ← 학급 운영
├─ 교육과정/
│  ├─ 학사일정 · 시간표 · 진도표 · 주간학습안내   ← 계획 원장
│  ├─ 단원 · 수업일지 · 행사 · 성취기준          ← 수업 실체
│  └─ 모아보기                                  ← Bases 보기
└─ 보고서 · 내보내기 · 백업
```

## 0. 처음 한 번 — 설치 직후 (10분)

1. `학급 공간 초기화` — 위 폴더 구조와 홈 노트를 만듭니다
2. **학급·학기 추가 및 전환** — 학년도·학기·학년·과목 목록 입력
3. 학급 메뉴의 **학생 추가·명렬표** → CSV 명렬표 가져오기 — 번호·이름 명단 등록 (미리보기 후 확정)
4. (선택) **AI 협업 설정** — 생기부 초안 등 LLM 협업을 쓸 때만. 기본은 꺼짐·익명화

## 1. 학년 시작 전 (2월 말) · 2학기 준비 (8월 중순) — 반나절

> 완료 지표: 교육과정 운영 뷰 설정 4종 모두 ✓ · 시수 점검 전 과목 '적정' · 일체화 간트에 전 단원 표시 · 모든 수행평가가 진도표 과제 열에 링크.

**교육과정 운영** 뷰의 체크리스트를 위에서 아래로 채우면 됩니다.

| 순서 | 할 일 | 방법 |
|---|---|---|
| 1 | 학사일정 | `학사일정 노트 열기` → 학기 시작·종료, 휴업일, 행사 표 입력. 수업일수가 자동 계산됩니다 |
| 2 | 기준 시수 | `기준 시수 노트 열기` → 구분(교과/창체)·과목별 연간 시수. **행 순서가 시수 점검 표시 순서**이며, 학교자율시간 과목·활동도 여기에 행을 추가해 등록합니다 |
| 3 | 기초시간표 | `기초시간표 노트 열기` → 요일×교시 과목 배치 |
| 4 | 진도표 | `진도표 차시 가져오기` — 엑셀·한글 표를 과목별로 붙여넣기 |
| 5 | 자동 배정 | `진도 자동 배정` → 시수 점검에서 부족·잉여 0 확인. 행사와 겹치는 차시는 시간표 칸 우클릭 → **진도 고정**(📌) |
| 6 | 일반 단원 | `일반 단원 일괄 생성` — 진도표를 단원별로 묶어 노트 초안 생성(기존은 건너뜀). 지도서 전문은 노트에서 수기 보완 |
| 7 | 프로젝트 단원 | `새 통합 단원 설계` — 개념 렌즈→스트랜드→일반화→GRASPS. 이관되는 차시는 일반 단원 전개에 `→ 통합 운영` 표시로 남습니다 |
| 8 | 평가 계획 | `평가 계획 가져오기` — 학교 평가계획 표 붙여넣기(시기\|단원\|평가 요소\|평가 기준\|평가 방법). 날짜는 그 주 배정 차시로 자동 매칭, 과제 노트+단원 연계+진도표 기입까지 한 번에 |
| 9 | 성취기준 | `성취기준 노트 생성` → `진도표 성취기준 링크화` — 코드가 클릭 가능한 [[링크]]가 됩니다 |
| 10 | 연계 마감 | `행사 노트 일괄 만들기` · `일체화 Bases 보기 만들기` |
| 11 | 전체 검토 | `일체화 간트 열기` — 단원 배치의 겹침·공백, ◆(과제) 몰림 확인 |
| 12 | 첫 주 | `주간학습안내 생성 (이번 주)` |

> 💡 LLM(에이전트) 협업이 가능하면 6·8·9는 `/unit-scaffold`·`/assessment-import`·`/standards-sync` 스킬이 지도서 전문·루브릭 전사까지 더 깊게 해 줍니다. 플러그인 명령은 LLM 없이도 되는 단독 경로입니다.

## 2. 매일

**아침 (1분)** — 오른쪽 **오늘 패널**: 학사 상태·시간표·미체크 출결 확인 → `오늘의 출결 체크`. 다가오는 행사를 눌러 준비물 확인.

**수업 변경 (10초)** — 시간표 칸 **우클릭**(모바일: 길게 누르기) → 과목 즉시 변경 / 수업 삭제 / 변경 제거. 저장 즉시 영향 과목의 진도가 자동 재배정됩니다.

**수업 후 (30초)** — 교시 우클릭 → `수업일지 쓰기`(날짜·교시·과목·진도 내용이 미리 채워짐, 단원 자동 제안). 관찰이 있으면 차시 인스펙터의 **학생부 근거** 버튼(학생 검색 → 단원·수업일지 프리필).

**수시** — `학생 빠른 기록`(관찰·상담·칭찬) · `과제 체크` · `할 일 빠른 수집` · `가정통신문 회신 체크`.

**자동으로 되는 일** — 지난 날짜의 수업일지는 다음 날 RAW로 확정되어 기록이 보존됩니다. 학기 경계가 지나면 학기 전환을 알림으로 제안합니다.

## 3. 매주 금요일 (10분)

1. **교육과정 운영** 뷰에서 다음 주 시간표 점검 — 행사 날 수업 변경, 겹치는 차시 📌 확인
2. `주간학습안내 생성` — 다음 주 안내문이 표로 생성됩니다
3. `일체화 간트 열기` — 이번 주가 단원·평가 흐름 어디쯤인지 확인 (오늘 선)

---

## 4. 평가하는 날

1. 진도표나 간트의 **◆**를 눌러 과제 노트로 이동 — 루브릭(◎○△)·문항별 채점 기준·평가지 딥링크가 노트 안에 있습니다
2. `과제 체크`(또는 노트의 확인표 직접 편집) — 제출/미제출/보완 + 메모. 확인표 아래 평가 정보 섹션은 다시 저장해도 보존됩니다
3. 단원 연계 과제는 저장 시 진도표 과제 열에 자동 기입됩니다

---

## 5. 달마다 · 수시

- **루틴 체크리스트** — 일간·주간·월간 반복 업무를 날짜별로 체크
- **가정통신문** — `가정통신문 회신 체크`로 미회신·확인 필요 학생 추적 (GTD 할 일 보드의 대기 열에도 표시)
- **분석·보고서** — 기간·학생별 기록 분포, 학생부 근거 커버리지(영역별 누락 보기)
- 손으로 만든 수업일지가 있다면 `수업일지 진도표 역링크 채우기` — 진도표 비고에 링크가 채워집니다
- **백업·유지관리** → `지금 백업 만들기` — 진도 자동 배정 같은 대량 변경 전에는 플러그인이 자동 스냅숏을 남기지만, 월 1회 수동 백업을 권장합니다

---

## 6. 학기말 (7월 · 12~1월)

1. **학생부 근거 검토** — `근거 검토 상태`에서 raw→검토 승격(교사 판단). 커버리지에서 영역별 누락 보완
2. **분석·보고서**에서 학기 요약 확인, 필요하면 Markdown 보고서·CSV 저장
3. **생기부 초안(선택)** — AI 협업 내보내기(자동 익명화) → `AI 결과/`에 생성된 초안을 검토 후 확정. 에이전트 협업이면 `/record-draft` 스킬(모드 A — RAW는 읽기만). **초안은 참고일 뿐, 판단과 확정은 항상 교사가 합니다**
4. **마감** — `지금 백업 만들기` → (선택) 보존 정리로 오래된 관리 파일 정돈 → 방학
5. 새 학기가 되면 학기 전환 제안 알림을 따라 설정의 학기를 바꾸고 **1번(학기 준비)**으로

## 7. 문제가 생기면

| 증상 | 해결 |
|---|---|
| 데이터가 안 보임/이상함 | **학급·데이터** → 진단 실행 (중복 번호·깨진 링크·필수 폴더 검사) |
| 파일을 지웠다/잃었다 | **백업·유지관리** → 최근 백업에서 누락 파일 복구 (기존 파일은 절대 덮어쓰지 않음) |
| 대량 작업을 되돌리고 싶다 | `백업/` 폴더의 `… 자동` 스냅숏에 변경 전 원본이 있습니다 (백업 정보에 계기 기록) |
| "구버전 폴더 이름" 알림 | 옛 백업을 복원한 경우입니다 — 알림의 안내대로 폴더 이름을 새 구조(학생 기록·단원·모아보기)로 변경 |
| 플러그인 업데이트 후 그대로임 | Obsidian 설정 → 커뮤니티 플러그인에서 껐다 켜기(재로드) |

## 부록 — 자주 쓰는 진입로

| 하고 싶은 일 | 가장 빠른 길 |
|---|---|
| 오늘 뭐 하지 | 오른쪽 오늘 패널 |
| 출결·기록·과제·할 일 | 왼쪽 학급 메뉴 "자주 쓰는 명령" |
| 이 교시 상세/기록 | 시간표 교시 좌클릭 / 우클릭 |
| 이 학생 최근 상태 | 대시보드 명단 클릭 → 학생 인스펙터, `학생별 타임라인 열기` |
| 학기 전체 흐름 | `일체화 간트 열기` |
| 모든 자료 검색 | `통합 목록 열기` (복합 필터·저장 가능) |
